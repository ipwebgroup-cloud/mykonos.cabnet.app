# Mykonos Inquiry Plugin v2.6.2 Handoff Focus Panel

## What changed
- added a read-only **Handoff Focus** panel to the **Internal** tab
- surfaces the three cues most useful during operator hand-off:
  - guest focus
  - queue checkpoint
  - current working picture
- helps the next assignee continue work without reconstructing context from raw fields

## Scope
- backend detail screen only
- no schema change
- no public theme change
- no list filter change
