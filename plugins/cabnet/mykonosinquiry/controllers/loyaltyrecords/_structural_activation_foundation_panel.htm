Structural activation foundation

This release is the first loyalty patch that ships the install-ready structural packet alongside the guarded workspace. It does not turn Loyalty Continuity into the live operational queue by itself. Instead, it stages the storage contract so activation can happen later in a narrow, explicit install step without destabilizing the Inquiry Queue.

  • adds the first install-ready loyalty table migration file
  • keeps the current workspace render-safe when tables are absent
  • frames the first activation as a controlled continuity layer, not a broad CRM expansion
  • preserves Inquiry Queue as the live production workspace until activation is deliberately executed
