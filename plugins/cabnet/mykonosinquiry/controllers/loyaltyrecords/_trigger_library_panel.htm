Trigger library

Loyalty Continuity should not start from a blank page. Define a narrow library of approved triggers so the first live wave is driven by clear reasons to re-engage rather than operator improvisation.

  • Seasonal return trigger: prior guest fit plus a credible next-visit window.
  • Milestone trigger: anniversary, repeat-stay timing, or a meaningful service memory worth acknowledging.
  • Referral trigger: strong satisfaction signals and a natural introduction opportunity.
  • Value-extension trigger: useful follow-up around services that deepen convenience or premium support.
  • Pause rule: if no clear trigger exists, do not force continuity. Hold in review or leave in Inquiry Queue.
