Loyalty Continuity daily workflow
Use this strip as the fast operator guide before widening the list, opening history, or stepping into deeper continuity diagnostics.
1. Read the continuity lane first
Start with source, continuity posture, latest outcome, packet state, next review, and ownership before opening the full workspace tabs.
  • Use the compact list for daily follow-up review.
  • Keep latest outcome and timing visible.
  • Open records only when the list signal says more context is needed.
2. Compare source inquiry against current continuity
Keep the source inquiry visible enough to explain why this guest sits in continuity, whether the lane came from queue work, or whether it was created manually.
  • Use source inquiry jumps when the original route matters.
  • Keep manual records clearly distinct from inquiry-backed lanes.
  • Use history only when the compact card suggests it.
3. Widen the list only for deeper diagnostics
The default list is for fast daily continuity review. Use List Setup and Workspace Docs when finish posture, reopen timing, queue watch, or stewardship detail needs a closer read.
  • Use List Setup for finish and reopen diagnostics.
  • Use Loyalty Help for continuity wording and route meaning.
  • Use Queue Help when a source inquiry decision still drives the lane.
