Operator quick start
Inquiry Queue daily workflow
Use this strip as the fast operator guide before dropping into filters, List Setup, or record-level actions.
1. Run the live queue pass
Start with the compact default view so ownership, urgency, and next follow-up stay visible before opening deeper record detail.
  • Scan owner and follow-up first.
  • Separate urgent or overdue rows early.
  • Keep list reading narrow before expanding history.
2. Decide whether work stays in queue or bridges out
Use the visible loyalty cues to tell whether the row is still queue-only, ready to transfer, already linked, or sitting in a draft or staged continuity posture.
  • Queue-only stays in active inquiry work.
  • Transfer-ready moves into continuity when mature enough.
  • Linked or draft rows can jump directly into loyalty review.
3. Open deeper review only when needed
Keep the list lean for daily use. Add heavier columns and deeper history only when the row-level signal shows something needs investigation.
  • Use List Setup for guest detail, source, and audit timing.
  • Use Queue Help for scan order and filter discipline.
  • Use Loyalty Help when continuity or handoff meaning is unclear.
